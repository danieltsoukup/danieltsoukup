- [Representative & Fair Synthetic Data](https://arxiv.org/abs/2104.03007) by Paul Tiwald, Alexandra Ebert & Daniel T. Soukup. Accepted as [ICLR 2021 workshop paper](https://sdg-quality-privacy-bias.github.io/).
